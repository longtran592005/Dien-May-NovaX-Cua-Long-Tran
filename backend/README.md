# Backend Microservices Skeleton

This folder contains the initial NestJS microservices skeleton:
- api-gateway
- auth-service
- catalog-service
- cart-service
- order-service
- payment-service
- ai-service

Each service exposes:
- GET /health

## Run locally
1. cd backend
2. npm install
3. npm run dev:gateway (or another service script)

## Build
- npm run build
